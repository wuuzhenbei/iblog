# LangChain 学习规划：从入门到精通

## 第一阶段：基础入门（1-2周）

### 1.1 了解 LangChain 核心概念
- 什么是 LangChain
- LLM 基础原理
- Chain、Agent、Memory 核心组件

### 1.2 环境搭建
- Python 环境配置
- 安装 langchain 及相关依赖
- API 密钥配置（OpenAI/Anthropic）

### 1.3 基础实践
- Hello World：调用 LLM API
- Prompt Template 使用
- Output Parser 初体验

---

## 第二阶段：核心组件（2-3周）

### 2.1 Models（模型）
- LLM 和 ChatModel
- 常见模型提供商集成
- 异步调用与流式输出

### 2.2 Prompts（提示词）
- PromptTemplate
- FewShotPromptTemplate
- ChatPromptTemplate

### 2.3 Chains（链）
- LLMChain 基础使用
- SequentialChain 链式组合
- LangChain Expression Language (LCEL)

### 2.4 Memory（记忆）
- ConversationBufferMemory
- Entity Memory
- 长期记忆与短期记忆

---

## 第三阶段：进阶应用（3-4周）

### 3.1 Agents（智能体）
- Agent 类型与选择
- Tool 使用与自定义
- ReAct 模式理解

### 3.2 RAG（检索增强生成）
- Document Loader
- Text Splitter
- Vector Store 与 Embeddings
- Retrieval 策略

### 3.3 工具与输出处理
- 自定义 Tools
- Function Calling
- JSON/CSV 输出解析

---

## 第四阶段：项目实战（4-6周）

### 4.1 初级项目
- 聊天机器人
- 文档问答系统
- 图片描述生成

### 4.2 中级项目
- 多代理协作系统
- 企业知识库问答
- 自动化工作流

### 4.3 高级项目
- 自主Agent系统
- 个性化AI助手
- 复杂业务自动化

## 学习建议

1. **动手实践**：每学一个概念立刻动手实验
2. **阅读源码**：深入理解内部实现
3. **参与社区**：活跃在 GitHub Issues 和 Discord
4. **项目驱动**：通过实际项目巩固知识
5. **持续关注**：LangChain 更新频繁，保持学习
